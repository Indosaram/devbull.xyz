## 정답

```
SELECT Department.dept_name, COUNT(Student.dept_id) AS student_number
FROM Student
RIGHT JOIN Department ON Student.dept_id = Department.dept_id
GROUP BY Department.dept_name
ORDER BY student_number DESC, dept_name

```
