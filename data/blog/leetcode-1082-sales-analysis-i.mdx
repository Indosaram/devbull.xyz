## 정답

```
SELECT seller_id FROM
(
    SELECT seller_id, SUM(price) AS total
    FROM Sales
    GROUP BY seller_id
) AS T1
WHERE total IN
(
    SELECT MAX(total) as max_total FROM ( 
        SELECT seller_id, SUM(price) AS total
        FROM Sales
        GROUP BY seller_id
    ) AS T2
)
```
