## 정답

```
SELECT project_id
FROM (
    SELECT project_id, COUNT(*) as count FROM Project
    GROUP BY project_id
) AS Temp1
WHERE count IN (
    SELECT MAX(count)
    FROM (
        SELECT COUNT(*) as count FROM Project
        GROUP BY project_id
    ) AS Temp2
)
```
