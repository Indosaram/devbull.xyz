## 단축키

| 내용 | 단축키 |
| --- | --- |
| 왼쪽 사이드바 열기 (각 버튼별 단축키가 별도로 존재합니다) | ⌘ + B |
| 전체 폴더 검색창 열기 | ⌘⇧ + F |
| 괄호 앞뒤로 이동하기 | ⌘⇧ + \ |
| 현재 줄 이동하기 | ⌥ + ↑ / ↓ |
| 현재 줄 복사하기 | ⌥⇧ + ↑ / ↓ |
| 현재 줄 삭제 | ⌘⇧ + K |
| 같은 단어 선택 | ⌘ + D |
| 현재 선택한 부분 모두 선택 | ⌘⇧ + L |
| 현재 선택한 단어 모두 선택 | ⌘ + F2 |
| 현재 줄 선택 | ⌘ + L |
| 위 / 아래에 커서 추가하기 | ⌥⌘ + ↑ / ↓ |
| 현재 파일을 화면 분할해 열기 | ⌘ + \ |

혹시 추천해주시고 싶은 자주 쓰는 단축키나 설정, 확장 프로그램이 있으시다면 아래에 공유해주세요 :)
