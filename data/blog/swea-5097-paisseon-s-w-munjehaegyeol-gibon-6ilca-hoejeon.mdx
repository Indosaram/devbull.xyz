# 문제

-   \[파이썬 S/W 문제해결 기본\] 6일차 - 회전

# 풀이

```python
T = int(input())
for test_case in range(1, T + 1):
    N, M = map(int, input().split())
    templist = list(map(int, input().split()))

    queue = []
    for i in range(N):
        queue.append(templist[i])

    for j in range(M):
        temp = queue.pop(0)
        queue.append(temp)

    print('#%d %d' % (test_case, queue[0]))
```
