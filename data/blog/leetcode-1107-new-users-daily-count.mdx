## 정답

```
SELECT login_date, COUNT(user_id) as user_count
FROM (
    SELECT user_id, MIN(activity_date) as login_date FROM Traffic
    WHERE activity = 'login'
    GROUP BY user_id
) as Temp
WHERE login_date >= date_add('2019-06-30', interval -90 day)
GROUP BY login_date
```
