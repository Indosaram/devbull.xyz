## Solution

```
SELECT product_name, unit
FROM Products
JOIN (
    SELECT product_id, SUM(unit) as unit
    FROM Orders
    WHERE order_date BETWEEN '2020-02-01' AND '2020-02-29'
    GROUP BY product_id
) AS Orders ON Products.product_id = Orders.product_id
WHERE Orders.unit >= 100

```
