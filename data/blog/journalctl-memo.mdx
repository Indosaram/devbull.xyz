## vim pipe

`vim` 으로 특정 키워드 검색하고 싶을 때

```
journalctl | vim - 

```

여기서 `vim` 의 `-` 옵션이 의미하는 것은 `stdin` 에서 입력을 받는다는 의미이다.

### vim 응용

만일 특정 키워드를 찾는다면 `/` 뒤에 키워드를 입력하고 엔터를 입력한다. 다음 매치를 찾으려면 `n` 을 입력한다.

만일 페이지 단위(화면크기)로 이동하려면 `Ctrl + F` 를 입력하고 `n` 또는 `b` 로 앞/뒤로 이동할 수 있다.

## 원하는 위치 보기

일반적으로 마지막 `n` 라인 보기를 제일 많이 사용한다.

```
journalctl -n 100
```

그냥 pager를 마지막에 두고 싶다면

```
journalctl -x -e
```

## 원하는 타임프레임 보기

`since` + `until`

```
 journalctl --since 2020-01-09 --until 2020-01-11

```

시간을 나타내는 단어도 사용가능하다.

```
 journalctl --since "-2hour" --until "10min"
```
