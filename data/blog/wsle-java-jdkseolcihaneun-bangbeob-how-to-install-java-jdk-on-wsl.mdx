Windows 운영체제에서도 리눅스 개발환경을 세팅할 수 있게 해주는 WSL은 매우 편리합니다. 리눅스 가상 머신을 세팅하거나 별도의 머신을 구매할 필요가 없이 윈도우 로컬에서 바로 리눅스와 파일 전송, 클립보드 공유 등이 가능하니까요.

다만 별도의 운영체제인만큼 개발환경 세팅도 각각 해주어야 하는데요. WSL에 JAVA JDK를 설치하는 방법을 알아보겠습니다. 먼저 WSL을 실행시킨 다음 `bash`에서 자바가 설치되어 있는지 확인한다.

```
java -version
```

오류가 발생하면 설치되지 않았다는 뜻이므로 아래 명령어를 입력해서 `openJDK`를 설치한다. 2020년 3월 기준으로 `JAVA 11`이 설치된다.

```
sudo apt install default-jdk
```

다운로드 되고 설치된 다음 화면에 `done.`이 출력되면 설치가 된 것이다. 처음에 입력한 명령어를 다시 입력하여 제대로 설치되었는지 확인한다.

```
java -version
```
