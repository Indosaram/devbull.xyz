# 문제

-   프로그래머스 : 코딩테스트 연습 - 완전탐색 - 카펫

# 풀이

```python
def find_denom(num):
    cand = []
    for i in range(1, int(num**0.5)+1):
        if num%i==0:
            cand.append([num//i, i])
    return cand

def solution(brown, red):
    redcand = find_denom(red)

    for cand in redcand:
        if 2*sum(cand)+4 == brown:
            return [2+cand[0], 2+cand[1]] 
```
